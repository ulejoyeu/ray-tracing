# RayTracing

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.10.

## Description

This web application can generate images using ray tracing, from a 3D scene described by 2 text files. The rendered image will be displayed in an HTML canvas.

These files describe respectively the scene and the screen.

You can find samples of these two files inside `src/data` folder. To generate the image, simply copy/paste their content into the corresponding text areas and click on the `Generate image` button.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
